# Setup Guide: Trustless Account & 2FA Portal

This single-page portal displays an account's credentials (email, password, and live 2FA code) directly to users based on a URL parameter (e.g., `index.html?id=1234`). All Supabase keys are hardcoded in the file, and the user sees no setup or connection prompts.

---

### Step 1: Create Your Supabase Project
1. Log in at [Supabase](https://supabase.com).
2. Click **New Project**, select an organization, name your project (e.g. `PortalDB`), and set a secure database password.
3. Choose the free tier and wait for provisioning to finish.

---

### Step 2: Create Table & Policies
We will establish a read-only permissions structure. Anyone with the public anonymous key can retrieve specific records, but they cannot edit or delete them.

1. Go to the **SQL Editor** tab in the left-hand menu.
2. Click **+ New Query**.
3. Paste this SQL script and click the green **Run** button:

```sql
-- 1. Create the credentials table
create table accounts (
  id bigint generated by default as identity primary key,
  created_at timestamp with time zone default timezone('utc'::text, now()) not null,
  email text not null,
  password text not null,
  two_factor_key text not null
);

-- 2. Turn on Row Level Security (RLS)
alter table accounts enable row level security;

-- 3. Allow public read-only access (SELECT) for anonymous users
create policy "Allow public read access"
on accounts
for select
to anon
using (true);

-- 4. Seed initial mock credentials
-- Note: the two_factor_key must be a standard base32 string
insert into accounts (email, password, two_factor_key)
values
  ('admin@company.com', 'SuperPass123!', 'JBSWY3DPEHPK3PXP'),
  ('marketing@social.io', 'MktgSafe2026', 'ORSXG5BRGAYDAMBQ');
```

---

### Step 3: Configure Your Code
1. Click **Project Settings** (gear icon in the bottom-left sidebar) -> **API** tab.
2. Copy the **Project URL** and the **Anon/Public API Key**.
3. Open `index.html` in your editor and input them at the top of the `<script>` tag:
   ```javascript
   const SUPABASE_URL = "https://your-project-id.supabase.co";
   const SUPABASE_ANON_KEY = "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...";
   ```
4. Save the file.

---

### Step 4: Access and Share Links
You can now access and distribute links to specific accounts using the row `id`:
- Link for Account 1: `index.html?id=1`
- Link for Account 2: `index.html?id=2`
- The user will immediately see the email, password, and live 2FA code updating every 30 seconds.
